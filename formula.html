Интересные факты

Ключевые формулы

Основные формулы дифференциального исчисления

  • Производная функции: f'(x) = lim(h→0) [f(x+h) - f(x)] / h
  • Производная суммы: (f(x) + g(x))' = f'(x) + g'(x)
  • Производная произведения: (f(x) * g(x))' = f'(x) * g(x) + f(x) * g'(x)
  • Производная частного: (f(x) / g(x))' = [f'(x) * g(x) - f(x) * g'(x)] / [g(x)]²

Основные формулы интегрального исчисления

  • Неопределённый интеграл: ∫f'(x) dx = f(x) + C
  • Определённый интеграл: ∫[a, b] f(x) dx = F(b) - F(a)
  • Интеграл степенной функции: ∫xⁿ dx = (xⁿ⁺¹) / (n+1) + C, n ≠ -1
